# Email Extractor
## _Only for educational purposes_

## Features

- Extracts Emails
- Save as file

## Installation

Javascript Version
```sh
cd EmailExtractor/JS
npm i fs
npm i readline
node main.js
```

Python Version (3.10.4)
```sh
cd EmailExtractor/Py
python main.py
```
